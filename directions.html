User Guide

This page will serve as a guide in how to navigate this website and use all of its features and functions. By referring to this guide the viewer should be able to successfully use this business website for its intended purposes of purchasing digital artwork or communicating with the business .

How to purchase content - In order to purchase content from this webpage the user must navigate to the Order Now Page. The user can achieve this two different ways.

  1. The first way is to simply click on the order now page directly from the homepage which will take them directly to the Order Now Page.

  2. The second way is a hidden access where if the user navigates to the actual product page and clicks on the picture of the product it will direct them to the Order Now Page.

Once on the page the user will be prompted to fill out a form that gives the user the option to fill in which products they would like to order and at what quantity. After the form is filled out the total will be displayed and the user will then be prompted to fill out another form with their mailing address and contact information.Once all this information is provided the user can complete their transaction by hitting the submit button on the form.

Contact Us- Some visitors might like to interact with us to request special orders or perhaps even leave feedback.and for that we have implemented a contact page. To access this page the user should navigate to the contact page via a link on the homepage. Once on the contact us page the user will be prompted to fill out a form that will attempt to send an email to the website's admin from the user's registered default email address.

Post a comment on our Blog - Since this is a community oriented website having a blog/message board seems like the most ideal way to let the artist communicate with the customers and community members. To access this page the user should navigate to the Blog Page. To comment on the message board the user should type their name and insert their message in the text box under the artist's posted blog. Please beware that before the comment is posted under the blog the server admin will review the comments and serve as a moderator to filter out any malicious comments. So please play nice!!!

Look at the Facts- This is a simple page the user can access by clicking on the Facts Page link on the homepage. This page will feature expected shipping times,frequently asked questions and our refund policies which is “ALL SALES ARE FINAL”.

Upcoming News- The News page can be accessed from the homepage and will feature upcoming products or changes that are coming to the website/business.

A Page About Us-The About Us page can be accessed from the homepage and will include a brief biography of the digital artist and some info about our business.

Need Directions? The page you are currently on! This page can be accessed from the homepage and includes this short instruction guide on how to navigate and use this website appropriately.
